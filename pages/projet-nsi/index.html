J'ai aussi appris à mieux utiliser le CSS pour changer les couleurs, les tailles de texte et l'alignement. Parfois, je ne comprenais pas pourquoi quelque chose ne fonctionnait pas, mais en relisant mon code j'ai trouvé mes fautes.
